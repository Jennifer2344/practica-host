🌐 Práctica 4: Hosting de Proyectos Web

Aprende a publicar tu sitio en Internet

📌 ¿Qué es un hosting?

Es un servicio que permite almacenar los archivos de tu sitio web y hacerlos accesibles desde cualquier parte del mundo a través de internet.

✅ Requisitos para subir un proyecto

  • Tu proyecto web con archivo index.html
  • Conexión a internet
  • Cuenta en una plataforma de hosting como Netlify o GitHub Pages

🚀 ¿Cómo subir tu proyecto?

  1. Ingresa a https://netlify.com
  2. Regístrate o inicia sesión
  3. Haz clic en "Add new site" > "Deploy manually"
  4. Arrastra tu carpeta del proyecto o el archivo index.html
  5. Netlify generará un enlace público

📤 Mandar el link

Una vez publicado, copia el enlace del sitio web y compártelo con tu profesor(a) o colócalo en tu entrega.

Ejemplo de link: https://proyecto-hosting.netlify.app
